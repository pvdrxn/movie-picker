# Movie Picker Mobile App - Project Mandates

This document outlines the core principles, technology stack, and educational goals for the Movie Picker project. All development must strictly adhere to these instructions.

## 1\. Project Overview

* **Goal**: Create a mobile application for picking movies.
* **Educational Focus**: Every addition and change must be explained in full detail. The goal is for the user to learn the technologies used in depth.

## 2\. Technology Stack (Strict)

Only the following technologies are allowed unless explicitly authorized by the user:

* **Frontend**: React Native (JavaScript)
* **Backend**: Python and Django Rest Framework
* **Database**: PostgreSQL
* **Environment/Tooling**: Expo for handling the Android emulator and development workflow.

## 3\. Development Guidelines

* **No External Technologies**: Do not use any libraries or frameworks outside of the ones listed above without prior approval.
* **Detailed Documentation**: For every code change, explain:

  * **Why** the change was made.
  * **How** the technology works (e.g., React Native components, Node.js middleware, SQL queries).
  * **Best practices** being followed.
* **Surgical Updates**: Maintain clean, idiomatic code that aligns with the established stack.
